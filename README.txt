Стек проекта:

1. ReactJS
2. React Router
3. HTML & SASS
4. JavaScript(ES6+)
5. Git
6. npm
7. Babel


Реализовано:

1. Логика игры в сапёра
2. Дизайн приложения
3. Роутинг по адресам
4. Три уровня сложности игры
5. Таймер игры в зависимости от сложности
6. Счётчик флажков
7. Обработка нажатия правой кнопки мыши (циклично меняются "флажки" и "знаки вопроса")
8. Автоматическая победа при открытии всех ячеек без мин
9. Всплывающее уведомление о победе и проигрыше
10. Кнопки перезапука игры и возврата на экран настроек
11. Обработка ошибок
12. Тестовый режим для быстрой проверки в процессе разработки сценария победы
13. Адаптация для устройств с разными диагоналями экрана


В разработке:

1. Создана страница рейтинга попасть на которую возможно нажатием на кнопку Rating.
2. На странице Settings добавлена область, но не реализована логика для ввода никнейма игрока, чтобы в дальнейшем по сценарию победы добавлсять игрова в рейтинг


P.S. Тестовый режим можно включить следующим образом:

1. Пройти по пути minesweeper_game/src/styles/common.sass
2. В самом конце файла раскомментировать код стилей test mode

Теперь при запуске игры, ячейки с минами будут помечены красным цветом
